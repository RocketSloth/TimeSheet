# Screenshots

Place your application screenshots in this directory:

- `main-interface.png` - Main TimeSheet interface showing the clock-in status, daily hours, and todo list
- `manual-entry.png` - Manual time entry section expanded view

## Adding Screenshots

1. Take screenshots of your application
2. Save them as PNG files in this directory
3. Name them according to the files referenced in README.md
4. Commit and push to GitHub
